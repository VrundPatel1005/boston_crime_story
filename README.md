# Boston Crime Story

## Overview
This project explores shooting-related incidents in Boston using the Boston Open Data Portal.  
It provides an interactive data story with **two Plotly.js charts** to visualize:

1. **Shootings by District** – shows which districts report the most incidents.  
2. **Shootings by Hour** – reveals the time-of-day trends for shootings.  

Together, these charts highlight **spatial and temporal patterns** in Boston shooting incidents (2023–2024).

---

## Data Source
- [Boston Open Data Portal: Crime Incident Reports (2023–2024)](https://data.boston.gov/dataset)  

## Tools & Libraries Used
- **Python / Google Colab** – data collection and preprocessing  
- **Pandas** – data manipulation and aggregation  
- **Plotly.js** – interactive charting  
- **Google Gemini** – CSS/design framework  

---

## Files
- `asst3.html` – main HTML page with interactive Plotly.js charts  
- `district.csv` – aggregated shootings per district  
- `hour.csv` – aggregated shootings per hour  
- `district_hour.csv` – combined dataset for linking charts  
- `Boston_Crime_API_Assignment.ipynb` – Colab notebook for dataset assembly

## How to View
1. Open `asst3.html` in a web browser or host it via [GitHub Pages](https://pages.github.com/) for interactive viewing.
2. Interact with the charts to explore both **district-level** and **time-of-day** trends in Boston shootings.
